# ChefCommunity 
**Plataforma Social de Gestión de Recetas y Planificación Inteligente**

Proyecto Intermodular para el ciclo de Desarrollo de Aplicaciones Web (DAW).

## Descripción
ChefCommunity es un ecosistema digital diseñado para los amantes de la cocina que buscan optimizar su organización diaria. La plataforma permite gestionar recetas propias, planificar el menú semanal y generar listas de compras automáticas, transformando la experiencia culinaria en un proceso eficiente y social.

##  Funcionalidades Mínimas (MVP)
Estas son las características que se implementarán en la primera fase del proyecto:
- **Gestión de Usuarios:** Registro, inicio de sesión (JWT) y perfiles personalizados con "Libro de Cocina Virtual".
- **CRUD de Recetas:** Sistema completo para crear, visualizar, editar y eliminar recetas con ingredientes detallados.
- **Buscador con Filtros:** Búsqueda por nombre de plato y categorías básicas (postres, ensaladas, etc.).
- **Planificador Semanal:** Calendario visual para asignar recetas a los distintos días de la semana y tomas (comida/cena).
- **Generador de Lista de Compra:** Extracción automática de los ingredientes necesarios según la planificación semanal.

##  Features Extra (Fases Posteriores)
- **Buscador "Lo que tengo en la nevera":** Filtro inteligente basado en ingredientes disponibles.
- **Consolidación de Cantidades:** Suma automática de ingredientes repetidos en la lista de compra.
- **Interacción Social:** Sistema de valoraciones (estrellas) y comentarios en recetas de terceros.
- **Exportación de Datos:** Generación de lista de compra en formato PDF o envío por WhatsApp.
- **Diseño Responsive Avanzado:** Optimización total para dispositivos móviles (uso en supermercado).

##  Stack Tecnológico (Propuesto)
- **Frontend:** React,
- **Backend:** Python flask
- **Base de Datos:** MySql
